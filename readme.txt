<strong>Link Designer Lite - Free Link Designer Plugin for WordPress</strong>

<strong>Link Designer Lite</strong> is a free WordPress plugin that allows you to style up your links with beautiful CSS3 effects. Packed with 12 cool animation templates, you can design your site’s link buttons or hyperlinks of any post in an elegant and eye catching fashion. 

It is an easy to use plugin that comes with shortcode generators that generates shortcodes for each of the existing as well as newly added links. Besides, the plugin comes with different customization possibilities to configure your site’s link as per your preference. It contains unlimited color options for adding/editing your site’s link background color, typography options for editing the link text fonts, font size and text transformation and many more. 

== Feature List ==
<p>We have listed all the options available in the plugin.</p>
* 12 Beautifully Designed Cool Animation Templates
* Shortcode Generator
* Unlimited Color Scheme
* Selected Typography Options
* Option to Set the Alignment of your Link Text
* Set the Link Target (New Tab or Same Page)
* Option to Make your Link Downloadable

== Installation ==
1. Unzip [plugin].zip
2. Upload all the files to the /wp-content/plugins/link-designer-lite
3. Activate the plugin through the \'Plugins\' menu in WordPress.
4. To add a custom link, go to any post/page and link on the Link Designer button besides the Add media button
